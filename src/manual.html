Teil A · Einstieg

1 · Über diese App & Was ist neu

Der TeamTalk VoiceOver Client ist ein grafischer Client für das TeamTalk-Protokoll, der speziell für blinde und sehbehinderte Nutzer entwickelt wurde. Alle Bedienelemente sind für Screenreader optimiert:

  • macOS – VoiceOver mit Braillezeile (primäre Plattform)
  • Windows – NVDA, JAWS (vollständig unterstützt, Alt+Buchstabe-Shortcuts für alle Schaltflächen)
  • Linux – Orca (grundlegend unterstützt)

TeamTalk ist ein Sprachkommunikationsdienst ähnlich wie Mumble oder Discord. Der Client verbindet sich mit einem TeamTalk-Server, tritt Kanälen bei und ermöglicht Sprach- und Textkommunikation.

Die App unterstützt vier Oberflächensprachen: Deutsch, English, Français, Español. Die Sprache kann unter Einstellungen → Allgemein → App-Sprache gewechselt werden; die App startet dann neu.

3 · Installation & erster Start

macOS

  1. DMG-Datei öffnen (Doppelklick).
  2. "TeamTalk VO Client.app" in den Programme-Ordner ziehen.
  3. Beim ersten Start: Rechtsklick → "Öffnen" (einmalig wegen Gatekeeper-Warnung).
  4. Mikrofon- und ggf. Bildschirmaufnahme-Berechtigung erteilen wenn gefragt.

Windows

  1. EXE- oder ZIP-Datei herunterladen.
  2. ZIP-Inhalt in einen Ordner entpacken (z.B. C:\Programme\TeamTalk VO Client).
  3. TeamTalk VO Client.exe starten.
  4. Windows Defender / SmartScreen: "Trotzdem ausführen" wählen.

Linux

  1. AppImage oder tar.gz herunterladen und entpacken.
  2. AppImage ausführbar machen: chmod +x TeamTalk*.AppImage
  3. Starten: ./TeamTalk*.AppImage
  4. espeak-ng installieren falls noch nicht vorhanden: sudo apt install espeak-ng

Erster Start

Beim ersten Start öffnet sich das Hauptfenster. Um sich zu verbinden, öffnen Sie zunächst das Verbindungsfenster über Datei → Verbinden oder die Schaltfläche Verbinden in der Schnellaktionen-Leiste. Serververbindungen werden als Profile gespeichert und bei jedem Start wiederhergestellt.

App-Update

Über Hilfe → Nach Updates suchen prüft die App, ob eine neue Version verfügbar ist. Das Update-DMG (macOS) wird direkt in der App heruntergeladen und gespeichert – kein Browser-Zugang zum Repository nötig.

9 · Kanäle, Nutzerliste & Chat

Dies ist der Hauptbereich "Kanäle und Chat". Er gliedert sich in zwei Hälften:

Linke Hälfte: Kanäle & Nutzer

  • Kanalliste: Zeigt alle Kanäle des Servers. Format: Kanalname, Nutzer: N, Passwort: Ja/Nein. Doppelklick oder Enter tritt dem Kanal bei. Kanalpasswörter werden bei Bedarf abgefragt.
  • Nutzerliste: Zeigt alle Nutzer im aktuellen Kanal. Format: Nickname, Benutzername, Status. Doppelklick öffnet den Privatchat mit diesem Nutzer.

Die Schaltfläche Kanal beitreten (Alt+K) tritt dem markierten Kanal bei.

Rechte Hälfte: Chat

  • Chatverlauf: Mehrzeiliges Textfeld mit Kanal- und Privatnachrichten.
  • Eingabefeld: Nachricht eintippen, dann Enter oder "Senden" (Alt+S).
  • Privat-Checkbox (Alt+P): Nachricht als Privatnachricht an den markierten Nutzer senden.
  • Verlauf exportieren (Alt+E): Speichert den Chat-Verlauf als TXT-Datei.
  • Verlauf leeren (Alt+L): Löscht den angezeigten Verlauf nach Rückfrage. Wenn Verlaufsspeicherung aktiv ist, wird auch die gespeicherte Datei gelöscht.

Chat-Verlauf speichern (optional)

Wenn unter Einstellungen → Allgemein → Chat-Verlauf speichern aktiviert, werden Nachrichten pro Server in einer JSON-Datei gespeichert (max. 200 Einträge). Beim nächsten Verbinden mit demselben Server wird der Verlauf automatisch geladen.

Der Verlauf kann jederzeit über "Verlauf leeren" zurückgesetzt werden.

10 · Benachrichtigungen

macOS – Native Benachrichtigungen

macOS-Systembenachrichtigungen für Kanal-Ereignisse erscheinen nur dann, wenn der betroffene Nutzer im selben Kanal wie Sie ist. Das reduziert irrelevante Meldungen auf vollen Servern.

Benachrichtigungen können zentral über Einstellungen → Anzeige → Benachrichtigungen aktivieren ein- und ausgeschaltet werden.

EreignisBenachrichtigung (Beispiel)
Nutzer betritt Ihren Kanal"Florian ist beigetreten – Kanal Lounge"
Nutzer verlässt Ihren Kanal"Florian hat verlassen – Kanal Lounge"
Neue PrivatnachrichtAbsendername + Nachrichtenvorschau
Um Benachrichtigungen zu deaktivieren: Einstellungen → Anzeige → Benachrichtigungen aktivieren ausschalten. Alternativ können Sie sie global in Systemeinstellungen → Benachrichtigungen → TeamTalk VO Client deaktivieren.

Sound-Ereignisse

Zusätzlich zu nativen Benachrichtigungen kann für jedes Ereignis ein eigener Klang abgespielt werden. Die Konfiguration erfolgt unter Einstellungen → Sound-Ereignisse (→ Abschnitt 18).

Benachrichtigungs-Regeln NEU ab v7.1

Über Einstellungen → Benachrichtigungs-Regeln… steuern Sie fein-granular, welche Ereignisse TTS, Klang, beides oder nichts auslösen. Regeln greifen von spezifisch nach allgemein: Eine Nutzer-Regel schlägt eine Kanal-Regel, diese schlägt eine Server-Regel, diese schlägt eine globale Regel.

Felder einer Regel

FeldBeschreibung
EreignisWelches Ereignis diese Regel betrifft: alle, Nutzer betritt/verlässt Kanal, Kanal-Chat, Privatnachricht usw.
Stichwort NEU v7.4Optional: Die Regel greift nur wenn die Nachricht dieses Wort enthält (Groß-/Kleinschreibung egal). Nur bei Nachrichten-Events aktiv.
GeltungsbereichGlobal, Server, Kanal oder Nutzer
WertName des Servers / Kanals / Nutzers (bei Global leer lassen)
AktionTTS + Klang, nur TTS, nur Klang, oder Stumm (alles unterdrücken)

Beispiele

  • Alle Chat-Nachrichten außer in Kanal "Lounge" unterdrücken: Globale Regel "Chat → Stumm", dann spezifischere Regel "Chat, Kanal: Lounge → TTS + Klang".
  • TTS nur wenn Stichwort "wichtig" vorkommt: Regel "Chat, Stichwort: wichtig → TTS". Ohne Stichwort in der Nachricht wird kein TTS ausgelöst.
  • Privatnachrichten von Nutzer "Max" immer vorlesen: Regel "Privatnachricht, Nutzer: Max → TTS + Klang".

Reihenfolge und Priorität

Regeln werden von oben nach unten durchsucht. Die erste passende Regel gewinnt. Spezifischere Regeln (Nutzer vor Kanal vor Server vor Global) sollten daher weiter oben stehen. Mit "Nach oben"/"Nach unten" können Sie die Reihenfolge anpassen.

Teil D · Audio

11 · Audio-Einstellungen

Erreichbar über: Datei → Sound-Konfiguration → Audio-Einstellungen… oder Einstellungen → Audio

Mikrofon aktivieren

  1. Öffnen Sie Audio-Einstellungen.
  2. Wählen Sie Ihr Mikrofon aus "Eingabegerät" (Auswahlmenü).
  3. Wählen Sie "Ausgabegerät" (Lautsprecher/Kopfhörer).
  4. Klicken Sie Audio anwenden (Alt+N).
Nach dem ersten Verbinden müssen Sie Audio-Einstellungen einmal anwenden, damit das Mikrofon aktiv ist.

Sprachaktivierung (Voice Activation)

Die Sprachaktivierung überträgt automatisch, wenn Sie sprechen – ohne Tastendrücken.

EinstellungBeschreibung
Sprachaktivierung (Checkbox)Ein-/ausschalten (Alt+S)
Aktivierungspegel (0–100)Schwelle ab der Übertragung beginnt. Niedrigerer Wert = empfindlicher. Standard: 30
Nachlauf in ms (0–5000)Wie lange nach Stille noch übertragen wird. Standard: 0 ms

Push-to-Talk (PTT)

Im PTT-Modus wird nur übertragen, solange Sie einen Hotkey gedrückt halten.

  1. Aktivieren Sie "Push-to-Talk" (Checkbox).
  2. Klicken Sie Hotkey aufnehmen (Alt+H).
  3. Drücken Sie die gewünschte Taste (Standard: Leertaste).
  4. Klicken Sie erneut oder drücken Sie Enter zum Beenden.
PTT und Sprachaktivierung schließen sich gegenseitig aus – nur eines kann gleichzeitig aktiv sein.

15 · Aufnahme und Medien-Streaming

Erreichbar über: Bereich-Auswahlmenü → "Aufnahme und Medien"

Kanal aufnehmen

  1. Wählen Sie das Aufnahmeformat (WAV, MP3 16k–320k).

Konversationen automatisch aufzeichnen

Wenn aktiviert, wird jeder Nutzer der spricht separat aufgezeichnet:

  1. Haken bei "Automatisch aufzeichnen" (Alt+M).
  2. Zielordner wählen.
  3. Dateiname-Muster festlegen (Platzhalter: %Y%m%d-%H%M%S, #userid#, %username%).
  4. Format wählen und "Eigene Stimme mit aufnehmen" (Alt+E) nach Bedarf.

Wählen Sie die Quelle im Auswahlmenü "Streaming-Quelle":

Datei

  1. Mediendatei-Pfad eingeben oder Durchsuchen (Alt+D).

  3. Abspielen (Alt+A) – streamt die Datei in den aktuellen Kanal.
  4. Position (0–1000): Zahlfeld zum Springen innerhalb der Datei.
  5. Streaming-Lautstärke (25–400): 100 = normal, 200 = doppelt laut.

YouTube, SoundCloud, Twitch, Bandcamp, Vimeo, Mixcloud (via yt-dlp)

  1. Für YouTube/SoundCloud: Suchbegriff eingeben und Suchen (Alt+S) klicken.
  2. Ergebnis aus der Liste auswählen – der Link wird automatisch eingetragen.
  3. Oder direkt einen Link einfügen.
  4. Streamen (Alt+R) klicken.
yt-dlp muss installiert oder im App-Bundle enthalten sein. Für Twitch, Bandcamp, Vimeo, Mixcloud: Link direkt einfügen.

  1. Sender aus der vordefinierten Liste wählen oder per Suche (Alt+S) nach Sendern suchen.

  3. Webradio streamen (Alt+W) klicken.

Podcasts

  1. Podcast-Name suchen (Alt+S) oder Feed-URL direkt eingeben.
  2. Feed laden (Alt+F).
  3. Episode aus der Liste wählen und Episode streamen (Alt+E).

21 · Plattformunterschiede

macOS

  • VoiceOver-Navigation: VO+Pfeiltasten bewegen sich durch alle Elemente. VO+Leertaste aktiviert. VO+M für die Menüleiste.
  • Listen: Werden als "Liste, N Einträge" angesagt (VoiceOver-Patch aktiv). Pfeiltasten oben/unten navigieren.
  • Auswahlmenüs: Werden als "Auswahlmenü" angesagt. Pfeiltasten ändern den Wert.
  • Zahlenfelder (SpinCtrl): Pfeiltasten ändern den Wert; direkte Eingabe möglich.
  • Fokus-Shortcuts: Cmd+1–4 setzen Fokus auf Kanalliste, Chatverlauf, Eingabe, Protokoll.
  • Fenster schließen: Cmd+W minimiert in Tray. Cmd+Q beendet komplett.
  • Standardkürzel: Cmd+O (Datei öffnen), Cmd+N (neuer Client), Cmd+, (Einstellungen).
  • Mikrofon-Berechtigung: macOS fragt einmalig nach Mikrofonzugriff. In Systemeinstellungen → Datenschutz freigeben.
  • Braillezeile: Alle Steuerelementnamen und -rollen sind auf Deutsch benannt.
  • Benachrichtigungen: Native macOS-Benachrichtigungen für Kanal-Ereignisse (ab v6.1.6 kanalbasiert gefiltert).

Windows

  • NVDA/JAWS-Navigation: Tab/Shift+Tab navigiert, Enter/Leertaste aktiviert. Im Formularmodus navigieren Pfeiltasten innerhalb von Listen.
  • Tastaturkürzel: Alt+markierter Buchstabe aktiviert Schaltflächen/Checkboxen. Alle Schaltflächen haben eindeutige Kürzel pro Bereich.
  • Menü-Navigation: Alt öffnet die Menüleiste, dann Buchstaben.
  • Standardkürzel: Strg+O, Strg+N, F4 (Einstellungen).
  • Tray: Doppelklick auf Tray-Symbol öffnet das Fenster wieder. Rechtsklick öffnet Tray-Menü.
  • Windows Defender: Beim ersten Start ggf. Ausnahme hinzufügen.

Linux

  • Orca-Navigation: Ähnlich wie Windows. Insert+Tab für Elemente-Liste.
  • Audiogeräte: PulseAudio oder ALSA. Gerätename im Auswahlmenü wählen.
  • espeak-ng: Systemversion wird bevorzugt (sudo apt install espeak-ng).
  • Tray: Funktioniert mit kompatiblen Desktop-Umgebungen (GNOME mit AppIndicator-Extension, KDE).
  • Einschränkungen: Desktopfreigabe und Maus-Fernsteuerung können je nach Compositor eingeschränkt sein.
